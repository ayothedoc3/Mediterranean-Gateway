# 🏛️ Mediterranean Gateway - Cultural Tourism Platform

A comprehensive multilingual cultural tourism platform focused on Mediterranean heritage sites and authentic cultural experiences across the Middle East, Europe, and North Africa.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://mediterranean-gateway.streamlit.app/)

## 📦 **Installation**

### Prerequisites
- Python 3.7+
- pip package manager

### Local Setup
```bash
# Clone the repository
git clone https://github.com/ayothedoc3/Mediterranean-Gateway.git
cd Mediterranean-Gateway

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run mediterranean_gateway.py
```
